# Tipos de Dados 
Conforme o ECMAScript standard temos 9 tipos de dados

* Data Types 
  * Primitive / Primitive Value
  * Structural 
  * Structural Primitive 

## Primitivos

* String
* Number
* Boolean
* undefined
* Symbol 
* BigInt

## Estruturais

* Object 
  * Array
  * Map
  * Set
  * Date
  * ...
* Function

## Primitivo EStrutural  / Structural Roo Primitive

* Null
